# City Explorer

**Author**: Dana Huffman

**Version**: 1.1.0

## Overview

This app is designed to allow a user to input a city name and retrieve information about said city.

## Getting Started

- Run `npm install`
- Add your API key to .env.sample

## Architecture

Created using Node, React, Bootstrap, JavaScript, CSS, HTML, and Axios
